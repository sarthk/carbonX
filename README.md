# CarbonX 🌍

India's most detailed personal carbon 
footprint calculator.

## Live App
👉 https://carbonx.in

## Features
- 18 car brands, 77 models, 263 variants
  with real kmpl data
- 4-meal daily food tracker with 
  Indian meal database
- State-specific electricity calculator
  (24 Indian states, real emission factors)
- Flight trip builder with Indian routes
- Personalised reduction tips
- Delhi city transport alternatives

## Built With
HTML, CSS, JavaScript
